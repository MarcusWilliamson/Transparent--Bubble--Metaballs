User Guide: Use the left and right mouse buttons to move and rotate the camera, and the scroll wheel to zoom. To view the scene in VR,
click the "Enter VR" button at the bottom of the screen.

Description: This program uses metaballs/marching cubes with a transparent material to simulate bubbles. I based my metaball implementation
on the three.js docs example at: https://threejs.org/examples/webgl_marchingcubes.html, which uses three.js's built-in marching cubes
module. I used WebXR to enable the scene to be viewed in VR. Camera movement was implemented using three.js's OrbitControls.
Upon pressing "Enter VR," the user's XR Transform (which controls the camera in VR) position will be set to the camera position set by
the user. The material I used is a PhysicalMaterial with its transmission set to 1.0, which creates the transparent effect. The scene also
applies environment mapping to the material based on the background texture, which is a 360 degree HDRi. This environment mapping generates
a lighting effect on the bubbles without using light sources in the scene. The program should run on a desktop or laptop computer fine.
Have not yet tested with standalone/mobile VR.

Background image is a free image from: https://polyhaven.com/a/buikslotermeerplein
